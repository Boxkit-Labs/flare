## Core Philosophy: Real-World Performance & UX
1. **Large Scale Maintenance**: Complexity is the enemy. Use patterns that scale to hundreds of screens.
2. **UI Polishing**: Focus on "finishing touches"—micro-animations, robust loading states, and user-centric error handling.
3. **Cross-Platform Excellence**: Code should run flawlessly on Web, Mobile, and Desktop from a single codebase.
4. **Optimistic UI**: Update UI immediately and roll back on failure to ensure zero-latency feel.

## Coding Patterns Observed
- **Redux for Complex State**: Uses Redux patterns for multi-layered commercial state management.
- **Modular Architecture**: Groups code by business feature (e.g., `invoices`, `clients`).
- **Repository Abstraction**: Strong separation of API logic from business logic.
- **Theme Centralization**: Heavy use of a centralized theme system to ensure visual consistency across thousands of widgets.
